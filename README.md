# Trusted Research Feed

信頼できる情報を収集・保存・整理するための、ブラウザだけで動くリサーチツール。

---

## Demo

https://wakkyai1617-boop.github.io/my-news-story/

---

## 概要

政治・AI動向のニュースをRSSフィードから自動収集し、**ルールベースの信頼スコア**と**信頼根拠の説明**を付与して表示します。
気になった記事は保存でき、用途メモの管理や、テーマ別の **Research Board** への整理も行えます。

バックエンド・DB・AIは一切使わず、HTML / CSS / バニラJavaScript のみで構成された静的サイトです。

---

## 制作目的

- 日々のニュースが「どの程度信頼できるか」を自分で判断する習慣をつけるため
- 情報収集→保存→整理の一連のワークフローをブラウザ完結で行うため
- フレームワーク不使用のバニラJSでSPAパターンを実践するため

---

## 主な機能

| 機能 | 説明 |
|------|------|
| RSS収集 | 政治・AI動向の複数フィードをCORSプロキシ経由で取得・表示 |
| 信頼スコア | URLパターン・情報源・鮮度から0〜100のスコアを自動算出 |
| 信頼根拠の表示 | スコアの加点・減点理由を詳細モーダルに表示 |
| 次に確認すべきこと | 記事ごとにリサーチの次ステップを提示 |
| 記事の保存 | 記事をlocalStorageに保存（用途・メモ付き） |
| 用途管理 | あとで読む / YouTube候補 / X投稿候補 / note候補 / 深掘り候補 / 要確認 |
| Research Board | テーマ別のボードを作成し、保存済み記事を整理。ボードメモ付き |
| 保存データリセット | フッターのボタンで保存記事・ボードを一括削除 |

---

## 使用技術

- **HTML / CSS / バニラJavaScript**（ビルド不要）
- DOMParser（RSS/Atom解析）
- localStorage（記事保存・ボード管理）
- CORSプロキシ（allorigins / corsproxy.io / thingproxy）

---

## データ取得方法

ブラウザから直接RSSを取得するとCORSで弾かれるため、公開CORSプロキシを順にフォールバックしながら取得します。

```
allorigins.win → corsproxy.io → thingproxy.freeboard.io
```

フィードの追加・変更は `js/feeds.js` の `FEEDS` 配列を編集するだけです。

| カテゴリ | フィード |
|---------|---------|
| 政治 | Google ニュース（"政治"）, NHK 政治 |
| AI動向 | Google ニュース（"AI 人工知能"）, ITmedia AI＋ |

---

## 信頼スコアの考え方

信頼スコアは「この記事が正しい」という断定ではありません。
**根拠の見えやすさ・情報源の透明性の目安**です。

スコアは以下のルールベースで算出されます：

| 要素 | 内容 |
|------|------|
| ソース種別 | 政府公式 85 / NHK・研究系 75 / ITmedia 65 / 一般ニュース 50 / ブログ 40 |
| 一次情報性 | 政府・研究ソース = 一次情報とみなす |
| 鮮度ボーナス | 24時間以内 +5、7日以内 +2 |
| 鮮度ペナルティ | 30〜60日 -5、60日以上 -15 |
| 日付不明 | -5 |

スコアが高くても、必ず元記事を読み、他のメディアでも確認することをお勧めします。

---

## 注意点

- 公開CORSプロキシに依存しており、プロキシ側の障害時は取得に失敗することがあります（「更新」ボタンで再試行）
- 記事の本文は表示せず、見出し・要約・配信元リンクのみを表示します
- 保存データはブラウザのlocalStorageに保存されます。ブラウザのデータ削除時に消えます

## 起動方法

### 公開版（推奨）

ブラウザで以下のURLを開くだけで使えます。インストール不要です。

https://wakkyai1617-boop.github.io/my-news-story/

### ローカル版

`fetch` を使うため `file://` ではなくHTTPサーバー経由で開いてください。

```bash
cd projects/my-news-story
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```
